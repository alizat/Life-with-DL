# Life-with-DL
Pet projects that make use of deep learning libraries in any way!

Projects so far:
- Multiple classification on the [Fashion MNIST](https://github.com/alizat/Life-with-DL/blob/main/Fashion_MNIST.ipynb) images dataset
- Node classification with [Graph Convolutional Networks (GCNs)](https://github.com/alizat/Life-with-DL/blob/main/gcn_node_classification.ipynb) on the Cora citations dataset
- Sequential recommendation on the MovieLens-1M dataset with [SASRec](https://github.com/alizat/Life-with-DL/blob/main/sasrec_movielens.ipynb) & [BERT4Rec](https://github.com/alizat/Life-with-DL/blob/main/bert4rec_movielens.ipynb)

Stay tuned!
